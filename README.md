# Controle de Finanças Pessoais

Este é um aplicativo para controle de finanças pessoais desenvolvido com **Electron**, **React**, **TypeScript** e **Tailwind CSS**.

## Tecnologias Utilizadas

- **Electron**: Para criar a aplicação desktop cross-platform.
- **React**: Biblioteca para construção da interface do usuário.
- **TypeScript**: Superset do JavaScript que adiciona tipagem estática.
- **Vite**: Build tool rápida para desenvolvimento web moderno.
- **Tailwind CSS**: Framework CSS utility-first para estilização rápida.
- **Shadcn/ui**: Componentes de UI reutilizáveis (baseados em Radix UI).

## Pré-requisitos

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

## Como Rodar o Projeto

1.  **Instale as dependências:**

    ```bash
    npm install
    ```

2.  **Inicie o ambiente de desenvolvimento:**

    Este comando iniciará tanto o servidor de desenvolvimento do Vite quanto a janela do Electron.

    ```bash
    npm run dev
    ```

## Scripts Disponíveis

- `npm run dev`: Inicia o app em modo de desenvolvimento.
- `npm run build`: Compila o TypeScript e faz o build do projeto (Vite).
- `npm run lint`: Executa o linter para verificar problemas no código.
- `npm run preview`: Visualiza o build de produção.
